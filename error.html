Can't connect to qBittorrent

qBittorrent Desktop couldn't reach the Web UI. Make sure qBittorrent is running and the Web UI is enabled.

—
Checklist
  • qBittorrent is running on the remote machine / Docker container
  • Web UI is enabled in qBittorrent → Tools → Options → Web UI
  • The host and port match what's configured in Settings
  • Your PC can reach the remote IP (try ping)
  • No firewall is blocking the port
